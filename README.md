# Harmonee Wellness Website

This is the starter website for Harmonee Wellness.

## Files

- `index.html` — homepage
- `api/claude.js` — Vercel serverless function for Claude API calls
- `.env.example` — example environment variable file
- `privacy.html` — starter privacy page
- `disclaimer.html` — starter wellness disclaimer page
- `crisis.html` — starter crisis resources page

## Important

Do not put your Claude API key inside `index.html`.

Store it in Vercel as:

ANTHROPIC_API_KEY
